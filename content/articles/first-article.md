# Selamat Datang di Blog Saya

Ini adalah contoh artikel pertama yang ditulis dengan **Markdown**. Anda dapat menambahkan lebih banyak artikel dengan menyalin file ini ke folder `content/articles/` dan mengubah kontennya.

## Mengapa menggunakan Next.js?

- **Performansi Tinggi** – Rendering di server dan static generation.
- **Pengalaman Pengembang** – Hot reload, TypeScript, dan Tailwind CSS.
- **SEO Friendly** – Dapat mengatur meta tag per halaman.

## Langkah Selanjutnya

1. Tambahkan file Markdown baru.
2. Perbarui daftar artikel di `src/app/articles/page.tsx`.
3. Deploy ke Vercel untuk melihat perubahan secara langsung.
